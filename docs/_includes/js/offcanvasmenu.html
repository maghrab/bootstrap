Off-canvas Menu

Off-canvas menu allows you to add a menu to your page that pushes the screen when the menu button is triggered.

Example

Check the live demo by clicking on this link

Usage

The off-canvas menu can be used with either a static or fixed navigation bar

Menu Button

To attach a button to the menu, simple add class off-canvas to your button. The button should be placed in the navigation bar at the top of screen.

{% highlight html %} {% endhighlight %}

Menu container

A container must be added to represent the menu. It should be a division with class off-canvas-menu

Menu container in the code

Since the container is off-canvas and it is fixed, you must place it inside the body.

JavaScript

The following function must be added to your JavaScript file in order to initiate the feature

{% highlight js %} SetOffCanvasMenu(); {% endhighlight %}
